# MORENT - Modern Car Rental Dashboard

"A high-fidelity Car Rental Dashboard named MORENT, built with HTML and CSS from a Figma design. Features a modern, responsive UI for browsing cars, managing pickups/drop-offs, and viewing vehicle stats. Designed for a premium user experience in car booking and management."

![Project Preview](assets/command.png)

## 🚀 Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
- **Car Catalog**: Browse through a wide variety of cars with detailed specifications.
- **Booking Management**: Intuitive interface for selecting pickup and drop-off locations, dates, and times.
- **Interactive UI**: Hover effects, smooth transitions, and a clean, modern layout.
- **Search & Filters**: Advanced search bar with category and filter options.

## 🛠️ Built With

- **HTML5**: Semantic structure for better SEO and accessibility.
- **CSS3**: Custom styling with Flexbox and CSS Grid.
- **Figma**: Source of the original UI/UX design.

## 📂 Project Structure

- `index.html` - The main entry point of the application.
- `styles.css` - Global styles and layout configurations.
- `index.css` - Component-specific styles.
- `assets/` - Image assets, icons, and car photos.

## 🔧 Installation

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/sahilkhan643/morent_car_rental_dashboard.git
   ```
2. Open `index.html` in your favorite web browser.
